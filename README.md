# AI-Powered-Industrial-Safety-Intelligence-for-Zero-Harm-Operations
Zero-Harm is an AI platform that fuses gas sensor, permit-to-work, maintenance, and worker-location data to detect compound safety risks—like a hot-work permit issued during rising gas levels—before they escalate. Built for ET AI Hackathon 2026, it features a live geospatial risk heatmap and explainable, action-ready alerts.
